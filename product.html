概述

不间断电源（ UPS ）是用于连接客户设备或装置（负载）与市电电网供应之间的重要设备，其功能是保证持续和优质的不间断电源供应给负载。即使在大范围停电情况下，它仍能给负载供应一个预定时间的电源（后备时间）。此外，UPS与传统的供电系统比较（如：市电电源，发电机组等），UPS具有很多优点，它能够向负载提供稳压、稳频、干净的、优质的电源。
UPS是针对电源故障的全面保护——当长时间或短时间主交流市电中断时，UPS系统通过电池保证连续向负载供电。电池连接到UPS系统的整流器和逆变器的中间。正常运行时向负载供电的逆变器是由整流器输入电能的；一旦市电出现故障，电池会提供电能给逆变器；因此，负载的供电不会中断。然而，电池向负载供电是有一定时间限制的，如果需要更长的后备时间，我们建议使用柴油发电机组；在这种情况下，电池的供电时间必须满足市电暂时中断后到柴油发电机全部运转起来的这段空隙就足够了。

应用范围
=> 个人电脑
=> 小型 IT网络
=> 局域网 (LAN)
=> 工作站
=> 服务器
=> 电子收款机系统 (POS)
=> 数据中心
=> 工业 PLC
=> 收银机
=> 医疗电气设备
=> 应急设备 (照明/报警)
=> 电信设备
=> 电子商务 (服务器场所，ISP/ASP/POP)
=> 工业加工
